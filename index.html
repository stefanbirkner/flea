Flea

A minimalistic server for testing web sites with static content.

Flea is designed to be a very small Node.js script. You must install Node.js for running flea.

Flea was developed by Stefan Birkner and is published under Apache License 2.0.

Run flea

node flea.js

Configure flea

flea.js has a few configuration variables:
port
the web server's port
documentRoot
the directory from which flea will serve files
directoryIndex
the resource to look for, when the client requests an index of the directory by specifying a / at the end of the directory name
contentTypes
content types for specific filename suffixes
